# Source Directory Structure

This directory contains the source code for the WakaJob mobile application.

## Folder Structure

- `screens/`: Contains all the application screens.
    - `Splash/`: Initial splash screen.
    - `Auth/`: Authentication related screens.
        - `Login/`: User login screen.
        - `Signup/`: User registration screen.
        - `OTP/`: OTP verification screen.
    - `Dashboard/`: Main application dashboard.

## Naming Conventions

- **Directories**: PascalCase (e.g., `Login`)
- **Files**: PascalCase with suffix (e.g., `LoginScreen.tsx`)
- **Components**: PascalCase

## Team Collaboration

Please ensure that any new screens or components follow this structure. Document your changes in the respective directory's README if applicable.
